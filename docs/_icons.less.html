Iconography

  • stylesheet
_icons.less  

Uses IcoMoon - Free icon set as a baseline from https://icomoon.io/.

To use an icon, add the class icon and addition to the desired icon class.

To add a new icon:

  • Upload edc-icons.json to a new project on IcoMoon
  • Add new icon to the library
  • Generate Font
  • Under "Preferences" make sure the setting Encode and Embed Font in CSS is selected.
  • Update generated json file
  • Update this file with new icons and the embeded font
  • Commit changes
